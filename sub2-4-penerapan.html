BANGUN DATAR

Rumus Cepat Bangun Datar

Diagonal Persegi

Jika sisi persegi = s, maka diagonalnya:

\(d = s\sqrt{2}\)

Jika panjang = p, lebar = l, maka diagonal:

\(d = \sqrt{p^2 + l^2}\)

Contoh: p=6, l=8 ‚Üí d=10

Alas = a, kaki = r, maka tinggi:

\(t = \sqrt{r^2 - (\frac{a}{2})^2}\)

Contoh: a=6, r=5 ‚Üí t=4

BANGUN RUANG

Diagonal Ruang dan Bidang

Diagonal Bidang Kubus

Rusuk = s, diagonal bidang:

\(d_b = s\sqrt{2}\)

Diagonal Ruang Kubus

Rusuk = s, diagonal ruang:

\(d_r = s\sqrt{3}\)

Diagonal Ruang Balok

Panjang = p, lebar = l, tinggi = t

\(d = \sqrt{p^2 + l^2 + t^2}\)
