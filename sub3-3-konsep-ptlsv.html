VIDEO PEMBELAJARAN

Apa itu Pertidaksamaan?

Tonton video berikut untuk memahami konsep dasar pertidaksamaan linier satu variabel.

Poin Penting:

  • Pertidaksamaan menggunakan tanda <, >, ≤, ≥ (bukan =)
  • Memiliki tak hingga banyak penyelesaian
  • Dapat digambarkan pada garis bilangan
  • Perhatian khusus saat mengalikan/membagi dengan bilangan negatif!
SIMBOL

Mengenal 4 Tanda Pertidaksamaan

>
Lebih dari
Nilai di kiri lebih besar dari kanan

≤
Kurang dari atau sama dengan
Nilai di kiri ≤ nilai kanan

≥
Lebih dari atau sama dengan
Nilai di kiri ≥ nilai kanan

Lingkaran penuh: ≤ atau ≥ (nilai termasuk)
Lingkaran kosong: < atau > (nilai tidak termasuk)
PERBANDINGAN

Persamaan vs Pertidaksamaan

Persamaan

  • ✓ Menggunakan tanda =
  • ✓ Satu penyelesaian (biasanya)
  • ✓ Contoh: 2x + 3 = 7 → x = 2
  • ✓ Operasi: ±×÷ di kedua ruas
  • ✓ Tidak ada perubahan tanda

Pertidaksamaan

  • ✓ Menggunakan <, >, ≤, ≥
  • ✓ Tak hingga penyelesaian
  • ✓ Contoh: 2x + 3 < 7 → x < 2
  • ✓ Operasi ±×÷ di kedua ruas
  • ⚠️ Balik tanda jika kali/bagi negatif!

ATURAN PENTING!

Jika kedua ruas dikalikan atau dibagi dengan bilangan negatif, tanda pertidaksamaan DIBALIK.

Contoh: -2x < 6

Kedua ruas dibagi -2 → x > -3

Mengapa? -2x < 6 → x > -3 (cek: x = 0 → 0 < 6 ✓, x = -4 → 8 < 6 ✗)

RANGKUMAN

Intisari Konsep PtLSV

Tanda

< : kurang dari
> : lebih dari
≤ : kurang dari sama dengan
≥ : lebih dari sama dengan

Penyelesaian

Umumnya memiliki tak hingga penyelesaian yang dinyatakan dalam interval atau garis bilangan.

Aturan Negatif

Mengalikan/membagi dengan bilangan negatif membalik tanda pertidaksamaan.

Garis Bilangan

Bulatan kosong: <, >
Bulatan penuh: ≤, ≥
Arsir ke kanan/kiri sesuai arah.
