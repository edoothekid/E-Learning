PERBANDINGAN

Persamaan vs Pertidaksamaan

Persamaan

  • Menggunakan tanda =
  • Satu penyelesaian
  • Contoh: 2x + 3 = 7 ‚Üí x = 2
  • Operasi ¬± √ó √∑ di kedua ruas
  • Tidak ada perubahan tanda

Pertidaksamaan

  • Menggunakan <, >, ‚â§, ‚â•
  • Tak hingga penyelesaian
  • Contoh: 2x + 3 < 7 ‚Üí x < 2
  • Operasi ¬± √ó √∑ di kedua ruas
  • Balik tandajika kali/bagi negatif!
ATURAN PENTING!

Jika kedua ruas dikalikan atau dibagi dengan bilangan negatif, tanda pertidaksamaan DIBALIK.

Contoh: -2x < 6

Kedua ruas dibagi -2 ‚Üí x > -3

Cek: x = 0 ‚Üí 0 < 6 ‚úì, x = -4 ‚Üí 8 < 6 ‚úó
