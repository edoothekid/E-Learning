Menggambar dan menginterpretasi grafik dari persamaan y = mx + c

VIDEO PEMBELAJARAN

Cara Menggambar Grafik Garis Lurus

Tonton video berikut untuk memahami langkah-langkah menggambar grafik persamaan garis lurus.

Poin Penting:

  • Bentuk umum: \(y = mx + c\)
  • m = gradien (kemiringan garis)
  • c = titik potong sumbu y (ordinat saat x = 0)
  • Grafik garis lurus cukup ditentukan oleh 2 titik.
KONSEP DASAR

Bentuk Umum Persamaan Garis Lurus

\(y = mx + c\)

dengan m dan c adalah konstanta

Menunjukkan kemiringan garis:

Konstanta (c)

Titik potong dengan sumbu Y:

(0, c)

Garis memotong sumbu Y di y = c

Simulator Grafik Garis Lurus

Atur nilai gradien (m) dan konstanta (c) untuk melihat perubahan grafik secara real-time.

METODE

2 Cara Menggambar Grafik

1

Metode Tabel

  1. Pilih beberapa nilai x (minimal 2)
  2. Hitung nilai y = mx + c
  3. Buat tabel pasangan (x,y)
  4. Plot titik-titik pada bidang koordinat
  5. Hubungkan titik-titik menjadi garis lurus

Contoh: y = 2x + 1

x 0 1 2
y 1 3 5
2

Metode Titik Potong

  1. Tentukan titik potong sumbu Y: (0, c)
  2. Tentukan titik potong sumbu X: saat y = 0
  3. 0 = mx + c ‚Üí x = -c/m
  4. Plot kedua titik potong
  5. Hubungkan kedua titik

Contoh: y = 2x + 1

Titik potong Y: (0, 1)

Titik potong X: (-1/2, 0)

TITIK POTONG

Titik Potong dengan Sumbu Koordinat

Titik Potong Sumbu Y

\( (0, c) \)

Substitusi x = 0 ke persamaan

Titik Potong Sumbu X

\( \left(-\frac{c}{m}, 0\right) \)

Substitusi y = 0 ke persamaan

üí° Catatan:

Jika m = 0 (garis horizontal), tidak ada titik potong sumbu X (kecuali c = 0).

Jika garis vertikal (x = k), tidak memiliki bentuk y = mx + c.
