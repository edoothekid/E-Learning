Memahami Sifat-Sifat Bilangan Berpangkat

Eksponen adalah bentuk perkalian berulang. Dengan menguasai sifat-sifat dasar di bawah ini, kamu dapat menyederhanakan berbagai operasi matematika yang kompleks dengan jauh lebih mudah dan cepat.

Rangkuman Sifat Eksponen

1. Perkalian Eksponen

Jika bilangan pokok sama, maka pangkatnya cukup dijumlahkan.

\(a^m \times a^n = a^{m+n}\)
Contoh Soal: \(2^3 \times 2^4 = 2^{3+4} = 2^7 = 128\)

2. Pembagian Eksponen

Jika bilangan pokok sama, maka pangkatnya cukup dikurangkan.

\(\frac{a^m}{a^n} = a^{m-n}\)
Contoh Soal: \(5^{10} \div 5^7 = 5^{10-7} = 5^3 = 125\)

3. Pangkat Dipangkatkan

Jika bilangan berpangkat dipangkatkan lagi, pangkatnya harus dikali.

\((a^m)^n = a^{m \times n}\)
Contoh Soal: \((3^2)^3 = 3^{2 \times 3} = 3^6 = 729\)

4. Sifat Pangkat Nol

Semua bilangan (kecuali nol) jika dipangkatkan 0 hasilnya adalah Satu.

\(a^0 = 1\)
Contoh Soal: \(1.000^0 = 1\)
